# PASSWORD GENERATOR
Costruiamo un sistema di generazione di una password sulla base di tre fattori:
- **Nome**
- **Cognome**
- **Colore preferito**
<br>
<br>
Infine attraverso di essi generiamo una password come da esempio: **NomeCognomeColore21**
<br>
<br>
